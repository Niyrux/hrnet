HRnet

Prérequis

Node.js ≥ 18
React 18

Fonctionnalités

Création d’un employé
Liste des employés
Navigation entre les pages + filtrage des employés

Technologies

React
Redux
React Router
Vite

Installation

git clone (https://github.com/Niyrux/hrnet.git)
cd hrnet
npm install
npm run dev

Particularité

Ce projet inclut une modale personnalisée publiée sur npm :
npm install @niirux/hrnet-modal
